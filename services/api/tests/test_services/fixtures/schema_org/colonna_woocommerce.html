Ethiopia Yirgacheffe Konga Washed

From the Konga cooperative in Yirgacheffe. This lot is washed processed and dried on raised beds.

Tasting notes: jasmine, bergamot, lemon and peach.

A superb filter coffee that also works beautifully as a light espresso.
